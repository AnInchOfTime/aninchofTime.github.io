资料来源

  • Hexo|中文文档
  • Fluid’s blog

环境

Windows

前提条件

  1. 需要有个人的GitHub
  2. 建立至少一个存储库,储存库要求如下:
    • 输入 username.github.io 作为存储库名称。 将 username 替换为您的 GitHub 用户名。
    • 用户或组织名称包含大写字母，必须使用小写字母
    • 仓库的可见性为公开（Public
  3. 参考官方文档
